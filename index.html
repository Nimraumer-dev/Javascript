What is JavaScript?

JavaScript is a high-level programming language that runs in web browsers and on servers (via Node.js). It enables interactive behavior on websites — for example responding to user actions, updating content dynamically, validating forms, and communicating with servers.

Where JavaScript runs

  • In a browser (Chrome, Firefox, Safari, Edge).
  • On a server using runtimes like Node.js.
  • In other environments (desktop apps, mobile apps, and more).

Why we use JavaScript

JavaScript brings web pages to life — it lets developers modify the page structure (DOM), handle events (clicks, typing), perform calculations, and talk to servers (AJAX / fetch). Modern web development uses JavaScript for both frontend and backend work.

How to include JavaScript

JavaScript is placed inside <script> tags. Scripts can be placed inside the <head> or anywhere inside <body>. Placing scripts at the end of body is common to ensure the page content loads first.

Simple example (non-alert)

You can run a quick console message to verify JavaScript is active in the page. Open the browser DevTools (Inspect → Console) and you will see the message produced by console.log().
